## 🚀 The Vision
Beleza Natural is not just a website; it is a **Medical-Grade Aesthetic Interface** designed for the modern African woman. Built with a mobile-first philosophy in Port Harcourt, Nigeria, this platform bridges the gap between traditional hair care and futuristic clinical data analysis.

## 🛠️ Technical Architecture (The "Stack")
- **Frontend:** Next.js 15 (App Router) for high-speed performance.
- **Styling:** Tailwind CSS with a custom 'Luxury Emerald' glassmorphism design system.
- **Animations:** Framer Motion for premium, scroll-triggered user experiences.
- **Backend:** Supabase (PostgreSQL) with Row Level Security (RLS) for clinical data protection.
- **Development Environment:** Fully engineered on **Android OS** using Termux & Acode—proving that high-end software doesn't require high-end hardware.

## ✨ Core Features
- **[Live] Smart Diagnostic Quiz:** A deterministic engine that identifies hair porosity and scalp conditions.
- **[Live] Real-time Database:** Instant storage of client profiles via Supabase.
- **[Futuristic] AI Scalp Analysis:** (In Progress) Integrating computer vision to analyze scalp health via uploaded photos.
- **[Futuristic] Appointment Engine:** A real-time booking system with WhatsApp automation.
